# 性能指標記錄 (FYI - Performance)

> 參考：[@fyi.md](./fyi.md)

記錄專案的性能指標歷史數據：構建時間、測試覆蓋率、代碼質量、運行時性能等。

---

## 📊 指標類別

### 構建性能
- 構建時間
- 構建產物大小
- 構建產物分析

### 測試覆蓋率
- 單元測試覆蓋率
- E2E 測試覆蓋率
- 代碼覆蓋率趨勢

### 代碼質量
- ESLint 錯誤/警告數量
- TypeScript 類型檢查結果
- 代碼複雜度指標

### 運行時性能
- 首屏載入時間
- 路由切換時間
- API 響應時間
- 內存使用情況
