## 🎯 標準化格式

### 1. 一般文檔標準格式

```markdown
# Document Title

> 📋 **目的**：Brief purpose statement

**最後更新**：YYYY-MM-DD  
**維護者**：開發團隊

---

## 📋 目錄

- [Section 1](#section-1)
- [Section 2](#section-2)

---

## Section 1

[Content here]

---

**最後更新**：YYYY-MM-DD  
**架構版本**：v2.0（Git-like 分支模型，51 張資料表）  
**維護者**：開發團隊
```

### 2. Mermaid 圖表文件標準格式

```markdown
# Diagram Title

**維護者**：開發團隊

---

\`\`\`mermaid
[Mermaid code here]
\`\`\`
```
