## 🔍 如何查找舊文檔

如果需要查閱歷史文檔：

1. **在 Git 歷史中查找**：
   ```bash
   git log --all --full-history -- "docs/SRP-重構完成報告.md"
   ```

2. **在 archive/ 目錄查看**：
   ```bash
   ls docs/archive/
   ```

3. **還原歷史文檔**（如果需要）：
   ```bash
   git checkout <commit-hash> -- docs/archive/<filename>
   ```
