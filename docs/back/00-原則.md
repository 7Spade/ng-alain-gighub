依赖规则
Routes → Shared → Core
Routes 可依赖 Shared 和 Core（仅必要服务）
Shared 可依赖 Core
Core 不依赖 Routes 或 Shared（类型定义除外）
Facades → Services → Repositories → SupabaseService
   Component → Facade → Service → Repository → SupabaseService → Supabase
禁止反向依赖
Core 不依赖 Shared（类型定义除外，需用相对路径）
Shared 不依赖 Routes
Routes 不直接依赖 Core（通过 Facades 或 Shared Services）
数据流
   User Action      → Component (Routes)      → Facade/Service (Shared/Core)      → Repository (Core/infra)      → SupabaseService (Core)      → Supabase (Database)

類型轉換流程
Database (snake_case) 
  → Types (core/infra/types/)
  → Repository (自动转换)
  → Service (camelCase)
  → Models (shared/models/)
  → Facade (ReadonlySignal)
  → Component
