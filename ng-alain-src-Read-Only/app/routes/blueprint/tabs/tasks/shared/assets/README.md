# Shared Task Assets

此目錄用於存放任務共享層可重複利用的靜態資源：圖示、SVG、樣式片段、JSON 配置等。請遵循下列原則：

- 僅放置無程式邏輯的純資源檔案；若資源僅屬單一 feature，請存放於對應 feature 的 `assets/` 目錄。
- 檔名、路徑需清楚描述用途並符合專案命名規範，避免縮寫或模糊含義。
- 若為大型檔案（>200KB）或可能頻繁更新，須評估是否改為透過 CDN / Supabase Storage 管理並在此處存放對應 manifest。
- 新增資源時同步更新引用處與文件，並檢查是否已有現成資源可用，避免重複。
- 不得在此存放機密或個資；所有敏感資料應透過環境設定或受保護的資料庫取得。
